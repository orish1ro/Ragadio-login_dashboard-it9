Student Portal User Manual
Registration

Open the registration page to create a new account.

Enter your Full Name, Username, and Email.

Create a Password and type it again in the Confirm Password box.

Click the Register button; the system will alert you if the passwords do not match.

Login

Enter your Username and Password on the login page.

Click Login to go to the dashboard.

Use the link at the bottom if you need to go back to the registration page.

Dashboard and Logout

View the table containing ID, Name, Email, and Course information.

Click the Logout link at the bottom to return to the login screen.
